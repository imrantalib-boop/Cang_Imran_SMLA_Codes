# Cang_Imran_SMLA_Codes

MATLAB implementation of the **Supervised Machine Learning Algorithm (SMLA)** for fractional-order differential equations, developed during a research visit at Stellenbosch University under the supervision of Prof. Cang Hui.

## 📁 Repository Contents

| File | Description |
|------|-------------|
| `Example_1_App_Sol_SMLA.m` | SMLA solution for Example 1 |
| `Example_1_App_sol_Operational_matrices_approach.m` | Operational matrices method |
| `Example_1_App_sol_Spectral_tau_approach.m` | Spectral Tau method |
| `Example_1_Absolute_error.m` | Error computation |
| `Example_1_Loss_vs_neuron_count.m` | Training loss analysis |
| `ope_der.m` / `ope_derr.m` / `ope_Int.m` | Operational matrix generators |
| `ort.m` / `coefvec.m` / `funvec.m` | OSCP basis utilities |
| `matrix_descretize_points.m` | Collocation matrix builder |
| `weight.m` / `right_hand_matrix.m` | Helper functions |

## 🚀 Usage

Run `Example_1_App_Sol_SMLA.m` to reproduce the SMLA results for Example 1. All other examples in the paper follow the same structure.
